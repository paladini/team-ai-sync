# Roadmap

## Milestone 1: Public Action MVP

- JavaScript GitHub Action with Node 24 runtime.
- `sync-config.json` validation and safe relative path handling.
- File and directory sync with `overwrite`, `skip`, `exclude`, and `deleteOrphans`.
- Git branch push plus PR create/update per target repository.
- Unit tests, local git fixtures, CI, README, and example workflow.

## Milestone 2: Hardening

- Marketplace-ready release notes and immutable version tags.
- Better dry-run report artifacts.
- Optional configuration schema export.
- Integration smoke tests against disposable repositories.

## Milestone 3: Enterprise Adoption

- GitHub App token examples.
- Organization policy examples.
- Metrics/reporting summary for large repository fleets.
